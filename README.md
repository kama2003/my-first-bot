# My First Bot

Простой Telegram-бот на Python с использованием aiogram 3.

## Что умеет

- Отвечает на команду `/start` приветствием
- Повторяет любое отправленное сообщение (эхо-бот)

## Запуск

1. Установи зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Создай файл `.env` и добавь токен бота:
   ```
   BOT_TOKEN=твой_токен_здесь
   ```

3. Запусти бота:
   ```bash
   python bot.py
   ```

Токен можно получить у [@BotFather](https://t.me/BotFather) в Telegram.
